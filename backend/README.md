# Form Submission for Tech Events in Tamil Nadu


## Setup
1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/tn-tech-events.git
   ```
create a venv
and install requirements
```bash
pip install -r requirements.txt
```
2. **Environment Configuration:**
   Create a `.env` file and add necessary credentials (e.g., `DATABASE_URL`, `API_KEY`).
3. **Run the Application:**
   ```bash
   python app.py
   ```

## Features
- **Event Submission:** Streamlined form for tech organizers to list events.
- **Location Filtering:** Targeted support for cities across Tamil Nadu (Chennai, Coimbatore, Trichy, etc.).
- **Validation:** Built-in checks for date, time, and venue details.
